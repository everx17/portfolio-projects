# Portfolio Projects (Student)

This repository contains three student projects showcasing a range of skills across:
- Unity game development (C#)
- Django backend / web application development (Python)
- React Native mobile development (JavaScript)

Each project has its own folder with a detailed README (overview, tech stack, demo links, and known
limitations).

## Projects

### 1) FYP — Catantics (Unity)
A 3D physics-based game prototype about caring for a cat, combining wacky physics, short levels,
and educational collectables (cat care tips/facts).

- Folder: `fyp-catantics/`
- Demo video (YouTube): https://youtu.be/BM_OCwSriFg
- Demo video (Drive): https://drive.google.com/file/d/1hET_ml8yYH9msk1ZXBgurSyBJGFk-hes/view?usp=sharing
- Details: see `fyp-catantics/README.md`

### 2) Y3 Coursework — BetterBeSocial (Django / Backend focus)
A social web app coursework project focused on backend features: authentication, following,
posting, image uploads, REST API, and realtime chat (WebSockets).

- Folder: `cw-better-be-social/`
- Demo video (Drive): https://drive.google.com/file/d/1WiKNTmx2SBQozFwhNYc85Ey6gM5rpEny/view?usp=sharing
- Report (Drive): https://drive.google.com/file/d/11bKGGAzZ4pHAb7ZY1Wv9WVEMdsT4iIVC/view?usp=sharing
- Details: see `cw-better-be-social/README.md`

### 3) Y3 Coursework — Fridge Friend (React Native / Expo)
A mobile app prototype that helps users track ingredients and discover recipes using an external
API, with additional work on Firebase + local persistence.

- Folder: `cw-fridge-friend/`
- Demo video (Drive): https://drive.google.com/file/d/16pBQnsJLDeFl2MjnhnyLbrk0EMAc4d_x/view?usp=sharing
- Details: see `cw-fridge-friend/README.md`

## Notes
These are student projects submitted under coursework / project time constraints. Some features
